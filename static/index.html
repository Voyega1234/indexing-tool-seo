How to use — quick guide

First time (per client)

  1. + New client — enter a name and the sitemap URL.
  2. ⚙ Settings — Connect Google, pick the GSC property, Save. (Optional: upload the service-account key to enable one-click Request Indexing.)
  3. 1. Fetch / Crawl site — crawls the site and builds the page list + internal link graph.
  4. 2. Check index status — asks Google which pages are indexed and why. The table then fills with status, issues and fixes.

Reading the table

  • Type — use the Types filter to hide clutter (System / Parameter / Author…) and focus on Page / Blog / Product.
  • Index — indexed or not, with Google's reason (Discovered, Crawled…). Hover for the full text.
  • Issue — what's wrong (worst first); hover a pill for detail. Recommendation = what to do about it.
  • In / Out — internal links; 0 In = orphan. Click a number to see which pages link there.
  • Dates — Created / Modified / Indexed. Use Group by, the filters, and click any header to sort.

Fixing & requesting indexing

  • Work the issues: add internal links to orphans, consolidate duplicates, remove sitemap junk, improve thin pages.
  • For pages that are ready, tick them (or select all eligible) → 3. Request Indexing and/or Submit sitemap.
  • Request Indexing is a nudge, not a fix — fix the underlying issue first, then nudge.

Routine (ongoing)

  • ↻ Refresh data — re-crawl + re-check in one go; newly-indexed pages get a 🆕 tag.
  • ⟳ Re-run checkup — recompute the diagnosis instantly (no crawl), e.g. after a site fix.

Button cheat-sheet

  • 1. Fetch / Crawl site — build/rebuild the page list + link graph (first, and after site changes).
  • 2. Check index status — pull Google index status + reasons (needs a saved GSC property).
  • ↻ Refresh data — re-crawl + re-check together (routine health check).
  • ⟳ Re-run checkup — recompute the diagnosis only; instant, no network.
  • 3. Request Indexing — nudge the selected pages via Google (needs the service account).
  • Submit sitemap — ping Google with the sitemap URL.
  • ⚙ Settings — Google connection, GSC property, service account.
